What To Create In Copilot Studio

  • Name the agent something like IT Solutions Agent.
  • Give it instructions to help with passwords, VPN, printer issues, device setup, software install guidance, and escalation suggestions.
  • Add knowledge sources such as your IT FAQ, internal support docs, and troubleshooting pages.
  • Publish the agent, then open Channels and copy the token endpoint for your website or mobile app channel.
  • Paste that endpoint below or set COPILOT_STUDIO_TOKEN_ENDPOINT in your environment.
