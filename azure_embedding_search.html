Setup Notes

What this page expects

Environment Variables

GEMINI_API_KEY must be available to the FastAPI app. You can also set GEMINI_EMBEDDING_MODEL, which defaults to gemini-embedding-001. The backend uses those values to build the client and cache document embeddings in memory.

Backend Route

This UI sends your question to /gemini-embedding-search/query and receives a JSON response with the top 2 ranked documents plus their similarity scores.
